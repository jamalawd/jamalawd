### 💡 Philosophy

Ship beats perfect when you are willing to be the first user. Run what you build on a real problem this week, feel the rough edges, then improve it. When something is ready for others, the bar moves again: installation, defaults, errors, and naming have to work outside your own machine. Sharing is not the finish line, it is the second pass that turns a script into something people can actually run.
